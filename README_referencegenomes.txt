Files for CELLOseq:

For Mapping, Grouping, and Errorcorrection:
Mus_musculus.GRCm39.transcripts.repeats.fa generated by merging mm39 transcriptome reference, gencode.vM35.transcripts.fa.gz (available at https://www.gencodegenes.org/mouse/) and consensus sequences for Mus Musculus and Mus Muridae repeats, Mus_musculus_merge_muridae_mus.fa (available from RepBase)


For Flair Alignment and Quantification:
GRCm39.primary_assembly.genome.fa and annotation (available at https://www.gencodegenes.org/mouse/)

mm39_repeatmasker_ucsc.rmsk.gtf can be downloaded from UCSC table browser (https://genome.ucsc.edu/cgi-bin/hgTables)

Annotation needs to be merged with:
cat gencode.vM35.annotation.gtf mm39_repeatmasker_ucsc.rmsk.gtf > gencode.vM35.annotation_repeats.gtf

And sorted by genomic coordinate
sort -k1,1 -k4,4n -k5,5n gencode.vM35.annotation_repeats.gtf > sorted_gencode.vM35.annotation_repeats.gtf
